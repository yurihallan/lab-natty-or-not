# Título do Projeto Extremamente Aesthetic ;)
SpeakForMe: Uma dublagem de um texto com as IAs Generativas

## 📒 Descrição
O SpeakForMe foi construindo com o intuito de ajudar e aumentar a performance de dubladores em trabalhos rapidos e repetitivos

## 🤖 Tecnologias Utilizadas
- IA Generativa: **[Synthesia](https://www.synthesia.io/)** para criação de avatares de dublagem curta.

## 🧐 Processo de Criação
Usei o "Synthesia" para reproduzir o texto de um curta em formato de video. O software garantir qualidade profissional de dublagem e edição.

## 🚀 Resultados
O vídeo gerado é uma transformação no ramo de dublagem profissional. O mundo das IAs Generativas destacar seu potencial transformador em muitas areas que ainda estão sendo explorada.
